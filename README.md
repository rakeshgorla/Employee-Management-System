# Employee Management System (SQL)

##  Project Overview

The Employee Management System (EMS) is a SQL-based relational database project designed to efficiently manage employee information within an organization.

The system stores employee details, departments, salaries, bonuses, payroll, qualifications, and leave records while generating meaningful business insights using SQL queries.

---

##  Features

- Employee Information Management
- Department & Job Role Management
- Payroll Management
- Leave Management
- Qualification Tracking
- Salary & Bonus Management
